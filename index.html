End-to-End Financial LLM Inference System

Qwen-2 (0.5B) · LoRA Fine-Tuning · FastAPI Inference

Built and deployed an end-to-end Financial LLM inference pipeline by fine-tuning Qwen-2 (0.5B) on a proprietary financial instruction dataset using LoRA, followed by weight merging for standalone inference. Designed a FastAPI-based CPU inference service, Dockerized and deployed in production, handling real-world issues like cold starts, dependency resolution, model loading failures, and runtime crashes. Optimized inference latency (~5–20s for 100–250 tokens) through prefill–decode separation, reduced prompt tokenization overhead, deterministic decoding, and CPU quantization experiments.
